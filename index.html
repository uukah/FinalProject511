EC2

Amazon's EC2, Elastic Computer Cloud, is a web service that offers on-demand computing in a cloud environment. With EC2 users do not have to rely on building and maintaining their own equipment, Instead they can rent what they need and scale as they need. EC2 offers flexible payment plans and the ability to choose from multiple computing configurations (processors, memory, GPU etc.) as well as between different Linux and Windows Distributions.

S3

Amazon S3, Simple Storage Service is an object storage service. Objects are stored in what Amazon calls buckets.

Cloudfront

Amazon CloudFront is a content delivery network (CDN) service built for high performance, security, and developer convenience.
